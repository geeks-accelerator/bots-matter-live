# Contributing to Bots Matter Live

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Install dependencies: `cd api && npm install`
4. Copy environment: `cp .env.example .env`
5. Run dev server: `npm run dev`

## Development

- **API**: Express.js server in `api/`
- **Frontend**: Static HTML/CSS/JS in root
- **Skills**: ClawHub skills in `skills/`

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes
3. Test locally
4. Submit a PR with a clear description

## Code Style

- Use clear, descriptive variable names
- Add comments for complex logic
- Keep functions focused and small

## Developer Certificate of Origin (DCO)

By contributing, you certify that you have the right to submit the work under the MIT license.

Sign-off your commits: `git commit -s -m "Your message"`
